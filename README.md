This is a basic 2d pixel rpg template in gamemaker to help me make games. It includes basic systems such as:
- dialogue + branching
- object interaction
- player movement
- camera movement
- npc interaction + ai
- inventory system
- using items from inventory
- room transitions
- real-time battles
- turn based battle
- health xp levels
