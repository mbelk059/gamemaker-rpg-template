This is a basic 2d pixel rpg template in gamemaker to help make any rpg. It includes basic systems such as:
- dialogue + branching
- object interaction
- player movement
- camera movement
- cutscene system
- npc interaction
- inventory system 
- using items from inventory
- room transitions
- real-time battles
- health xp levels

future additions:

- menu system
- save and load
- dialogue portraits

- turn based battle
- quest log
- camera effects
- sound design
